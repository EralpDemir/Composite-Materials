Input of properties are done through PROPS utility
List of inputs:
PROPS(1): E11 (MPa)
PROPS(2): E22 (MPa)
PROPS(3): nu12 (-)
PROPS(4): G12 (MPa)
PROPS(5): theta0 (deg)
PROPS(6): theta1 (deg)
PROPS(7): phi (deg)
PROPS(8): a (mm)
PROPS(9): t (mm)
